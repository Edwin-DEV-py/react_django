1.instalar Django
2.Instalar djangorestframework para la api.
3.instalar django-cors-headers para asignarle quien puede entrar a nuestro backends
4.instalar coreapi
5.instalar gunicorn para desplegar
6.instalar whitenoise para los archivos estaticos
7.instalar psycopg2 para usar postgres

-----------react------------
1.npm i react-router-dom = para tener multiples paginas en el front
2.npm i react-hot-toast = modeulo para notificaciones personalizadas
3.npm i axios = para hacer peticiones
4.npm i react-hook-form = validar los campos desde el front

-crear las paginas que se van a mostrar
-crear cada componente por separado para tener un orden y poder llamarlos en distintas paginas
-crear la carpeta de api para las peticiones

-con npm run build creamos la carpeta dist
